    nmap -sC -sV -p21 10.129.42.253
        -A (dont care about detection)
        -O operating system
        -v verbose
        -p- all ports
        -T5 (0 to 5 timing, higher is faster)
        -sS ("half-open" "stealth" scan doesnt complete 3way handshake) this is default scans if sudo permission used
        script location "/usr/shar/nmap/scripts"
        -Pn (dont ping, just scan)
        -f or --mtu <number> (fragments packet to bypass firewall)
